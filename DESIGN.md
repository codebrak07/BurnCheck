# Design System - VoltAgent Inspired

This document captures the visual language, typography, colors, and layout rules for the **BurnCheck** interface, inspired by the VoltAgent design language.

## Colors

### Brand & Accent
- **Electric Green** (`primary`): `#00d992` (CTAs, live indicators, branding dots)
- **Primary Soft** (`primary-soft`): `#2fd6a1` (links and hover accents)
- **Primary Deep`** (`primary-deep`): `#10b981`

### Surface
- **Canvas (Background)**: `#101010` (the default near-black page surface)
- **Canvas Soft**: `#1a1a1a` (code blocks, text inputs, secondary cards)
- **Hairline Border**: `#3d3a39` (1px solid border on cards, inputs, and rows)

### Text
- **Ink**: `#f2f2f2` (default light gray text for body)
- **Ink Strong**: `#ffffff` (pure white text for high contrast headings)
- **Body**: `#bdbdbd` (secondary text for descriptions and paragraphs)
- **Mute**: `#8b949e` (captions, placeholders, and footer text)

## Typography

- **Sans-Serif (Inter)**: Display, titles, and body content.
- **Monospace (JetBrains Mono)**: Inline code snippets, numeric spend counters, and table data.

### Hierarchy
- **Display XL (Hero H1)**: 60px, weight 400, `-0.65px` tracking.
- **Display LG (Section H2)**: 36px, weight 400, `-0.9px` tracking.
- **Display MD (Card Title)**: 24px, weight 700, `-0.6px` tracking.
- **Eyebrow Mono**: 14px, weight 600, uppercase, `2.52px` tracking.
- **Body Default**: 16px, weight 400, line-height 1.65.
- **Code Block**: 13px, weight 400.

## Shapes & Radii
- **Buttons / Inputs**: 6px (`rounded-sm`).
- **Cards / Code Boxes**: 8px (`rounded-md`).
- **Status Tags / Pills**: 9999px (`rounded-full`).

## Layout & Separation
- **Section Dividers**: 1px dashed `rgba(79, 93, 117, 0.4)` separating page zones.
